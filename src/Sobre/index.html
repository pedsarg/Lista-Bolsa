Introdução

Este site foi criado como método avaliativo na disciplina Laboratório de Engenharia de Software e tem como objetivo ajudar os investidores a obterem informações confiáveis de forma simples.

Explicação sobre cada página do site:

  • Na página inicial, você encontrará informações gerais sobre as ações e será o ponto de partida para qualquer página do site.
  • O cadastro e login permitem que o usuário tenha informações personalizadas e interaja no fórum.
  • Na página de "Consultar Ações", é possível pesquisar e obter informações sobre uma ação de forma simples e fácil.
  • A página "Calculadora de Ganhos" permite simular ganhos ou possíveis perdas, bastando inserir os dados pedidos. O resultado será exibido logo abaixo!
  • Já na página "Fórum", foi elaborado um ambiente de interação entre os usuários, para compartilharem informações úteis e auxiliarem iniciantes.

Tecnologias Utilizadas

Para a criação deste site foram utilizadas as seguintes linguagens:

  • HTML5: Para a estruturação e formatação da página.
  • CSS3: Para a estilização e responsividade da página.
  • JavaScript: Para a criação de efeitos e interatividade na página.
  • Node.js: Para conexão com a API e com o banco de dados.
  • MariaDB: Para criação do banco de dados.

Foram utilizados os seguintes frameworks e bibliotecas:

  • Express.js: Para a criação da API e conexão com o banco de dados.
  • Bibliotecas: SQL e Axios.

Além disso, foram utilizados:

  • Visual Studio Code: Como editor de código.
  • DBeaver: Para a gestão do banco de dados.
  • Git: Para o versionamento do código.
